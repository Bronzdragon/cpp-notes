Plain Old Data is any data structure with no user-defined (copy) constructor or copy assignment operator and has only public members who themselves are POD (no functions, but pointers to functions).

No “magic” will go on (e.g. hidden vpointers, construction).

“a type that relies on the compiler-generated default constructor, has no virtual functions or virtual base classes or data member initializers, and all its bases and members satisfy the same restrictions."

Basically a C-struct.
